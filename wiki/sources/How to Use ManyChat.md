# How to Use ManyChat

> GoGet'Em Community full training on ManyChat. Gogo has earned $65,000 directly attributed to ManyChat automations. Covers account setup, keyword triggers, personalizing messages, CTAs, and buyer/seller post examples.

## Key Takeaways

- **$65,000 earned from ManyChat automations** — Gogo attributes this revenue directly to keyword-triggered DM sequences.
- **Account setup:** Connect Facebook Business Page + Instagram to ManyChat; free tier available but Pro unlocks full automation sequences.
- **Keyword triggers:** A post says "comment HOMEVALUE" → ManyChat detects the keyword → sends automated DM with lead magnet or CTA link. Works on Facebook and Instagram.
- **Personalization:** Messages can include the commenter's first name automatically, making automation feel human.
- **Buyer post example:** Post about searching for homes → comment "SEARCH" → ManyChat sends IDX link for MLS search.
- **Seller post example:** Post about home values → comment "VALUE" → ManyChat sends home valuation tool link.
- **CTAs drive conversions:** Every automated flow should end with a clear next step — book a call, fill out a form, click a link.
- **Sequence building:** Beyond single-message triggers, ManyChat can run multi-step sequences (quiz flows, drip campaigns) to qualify leads and expand email lists.
- Full training available in GoGet'Em Community.
