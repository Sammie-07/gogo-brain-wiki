# Create 3 Bios with ChatGPT (Gogo Bethke)

> Gogo's framework for creating three distinct bios — buyer/seller, agent attraction, and blended — each optimized for different contexts, then adapted for platform character limits using ChatGPT prompts.

## Key Takeaways

**The 3-Bio Framework:**
- **Bio A — Real Estate (Buyer/Seller):** Who you are, what you do, where, how long you've been in the industry. Used for listing appointments, seller-focused websites, buyer consultations. No agent attraction content.
- **Bio B — Agent Attraction:** Focused on your production, team structure, revenue share, and eXp value proposition. Used for recruiting conversations, agent-focused speaking, GoGet'Em Community promotion.
- **Bio C — Blended ("About Me"):** Combines A + B. Used for podcasts, speaking engagements, stage presence — anything where the audience is mixed.
- **Bio D — Platform-Optimized:** Take whichever bio is relevant, shrink it to fit platform character limits using ChatGPT.

**Platform character limits:**
| Platform | Character limit |
|----------|----------------|
| Instagram | 150 |
| Facebook | 255 |
| LinkedIn | 220 |

**ChatGPT workflow:**
1. Copy the provided ChatGPT prompt template (linked below the video)
2. Paste into ChatGPT → it asks you questions
3. Answer the questions → ChatGPT generates your bio
4. For Bio C: paste A + B into ChatGPT → prompt it to blend them
5. For platform bios: paste the master bio → ask ChatGPT to shorten to the platform's character limit
6. Store all versions in your **Brand Tracker** — one document, everything accessible forever

**Time investment:** ~1 day to complete all four bio versions.
