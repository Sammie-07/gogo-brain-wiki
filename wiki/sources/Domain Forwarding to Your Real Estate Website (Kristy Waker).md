# Domain Forwarding to Your Real Estate Website (Kristy Waker)

> How to forward your custom domain (e.g., christysrealestateteam.com) to your BoldTrail/KVCore website so visitors see your clean branded URL instead of the ugly eXp default. Part of *Gogo's Social Media Challenge 2026*.

## Key Takeaways

- eXp agents get a BoldTrail/KVCore website with a default URL like `firstname-demo.exprealty.com` — unusable for marketing.
- **Domain forwarding** redirects your clean custom domain to that website. Free — you've already paid for the domain.
- **Steps (GoDaddy):**
  1. Copy your BoldTrail/KVCore website URL
  2. GoDaddy dashboard → 9-dot menu → My Account → **Domains** → find your domain → three dots → **Forward**
  3. Paste your BoldTrail URL into the destination field → preview to verify
  4. Choose forwarding type: **Forwarding with Masking** (recommended)
  5. Add meta keywords (e.g., "homes for sale," "Howell," "Brighton") → Save
- **Forwarding types:**
  - **Permanent (301):** Sends user to destination; search engines index the destination URL
  - **Temporary (302):** Temporary redirect; search engines treat it as short-term
  - **Forwarding with Masking:** Sends visitor to destination but **keeps your custom domain in the address bar** — visitors never see the ugly eXp URL. Also allows meta tags for SEO.
- Works on any domain registrar (GoDaddy, Google Domains, Namecheap, Squarespace) — every domain platform supports forwarding.

## Quotes
