# How to Build a Funnel in GoHighLevel (Jonathan Trage)

> Step-by-step GoHighLevel funnel builder tutorial covering templates, page editing, global sections, mobile optimization, and SEO.

## Key Takeaways

- **3 funnel components**: Landing page (user lands on offer), Opt-in/checkout page (they give info or pay), Thank you page (confirmation + next steps)
- **Path**: Sub-account → Sites → Funnels → Create New Funnel → select template or start blank
- **Recommended**: use templates, browse by category, preview before selecting
- **Layer system** (color-coded): Green = section, Blue = row, Purple = column, Orange = element
- **Global sections**: save footer/header as global → changes in one funnel step sync to all → major time saver for multi-step funnels
- **Mobile mode**: always check desktop AND mobile — clone sections if layout differs between devices
- Font sizes can be set independently for desktop and mobile
- **Publishing**: saves reflect immediately on live URL; publish pushes changes to domain
- **SEO**: add title, description, keywords, social image before publishing
- **Checkout page**: add 1-step or 2-step order form elements; set next step (thank you page)
- **Domain**: connect GoDaddy or purchased domain in funnel settings
- Auto-save can be toggled on/off; always manually save before previewing

## Quotes

> "As you are editing your funnel, it's always good to save. I have the auto save off, but you can turn it on if you want."
