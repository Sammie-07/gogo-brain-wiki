# How to Edit Reels and Stories on Instagram (Gogo Bethke)

> Step-by-step walkthrough of Gogo's Instagram editing workflow for Reels and Stories — captions, filters, CTAs, location tags, mentions, and links. Part of the GoGo Social Media Challenge 2026.

## Key Takeaways

- **Gogo's honest framing:** "I don't do attention to detail. I just freaking push buttons." — this is a practical mechanics tutorial, not an advanced editing course.
- **Reels workflow:** + sign → swipe up to select video → bottom bar 2nd icon = captions
- **Caption layout:** Gogo prefers the 2nd layout option; always reposition away from clothing/face
- **In Reels, you cannot tag a person inside the editor** — must tag after posting
- **Filters (Reels):** 6th icon from left (star shape) → saved filters → Gogo uses "aesthetic" filter
- **CTA text:** Tap text → type message → use left sidebar slider to reduce to one row → drag to bottom of screen
- **Gogo's Reels checklist before posting:** captions ✓ filter ✓ CTA text at bottom ✓
- **Stories access:** Main screen → swipe left; tap = photo, hold = video recording
- **Stories features:** Swipe up from preview to access all editing tools
- **Location tag:** Add → pinch to shrink with two fingers → drag to corner (still fully tags the location)
- **Hidden tag trick:** Drag a mention or location tag past the bottom edge of the screen — it disappears visually but still counts as tagged. Don't drag toward the middle (trash can)
- **Stories captions:** Move away from face; position in lower-left corner
- **Stories link:** Add URL → resize small → reposition away from face
- **Four most important features:** location tag, person tag, website link, captions
- **Homework:** Create a fake practice video and test all four tagging features until they feel natural
