# Trello Board Tutorial (Alejandra)

> GoGet'Em Community tutorial: create and manage a Trello board from scratch including cards, checklists, automation buttons, power-ups, and team filters.

## Key Takeaways

**Board Setup:**
- Create board → name it → set visibility (workspace or public) → change background color
- Standard lists: Ideas / Projects To Do / In Progress / Finished (optional: In Revision)

**Card Setup:**
- Add cover image for visual recognition
- Enable "Watching" to receive email notifications for all card changes
- Click "Join" to be assigned to the card
- Add team members → assign individual tasks with separate due dates per member
- Description: document goals, expected results, links, attachments
- Checklist: paste from a document — Trello creates list automatically; name it exactly (e.g., "to-do") — spelling must match automation trigger

**Automation Buttons:**
- Add button on card → action: "Move card to [list]" → set list position (top)
- Create buttons for each transition: Ideas → Projects To Do, Projects To Do → In Progress

**Automation Rule (key):**
- Automations → Create Rule → Trigger: "When checklist named [to-do] is completed in a card by anyone" → Action: "Move card to top of Finished list"
- Checklist name must match EXACTLY — spelling matters

**Power-Ups (add in Power-Ups menu):**
- **Calendar**: see all tasks by month, shows due dates and completion
- **Card Aging**: inactive cards become transparent/faded — surfaces forgotten projects
- **Repeat Cards**: set any card to repeat (weekly, monthly) at specific time → auto-creates in chosen list → use for recurring tasks like monthly bookkeeping

**Team Management:**
- Filter by member to see only that person's tasks
- Collapse lists with no matching cards for cleaner view
- Alejandra uses Trello for GoGet'Em Community team board (Monday heads-up card repeats weekly)
