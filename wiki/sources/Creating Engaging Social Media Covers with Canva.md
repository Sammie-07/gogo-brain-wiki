# Creating Engaging Social Media Covers with Canva

> GSMC 2026 training. Walkthrough for creating branded social media cover images in Canva — required elements, QR code generation, multi-platform resizing, and download settings.

## Key Takeaways

- **Tool:** Canva. Search templates by platform (e.g., "Facebook cover") or niche ("real estate") to start from a relevant base.
- **Required elements in every cover:**
  - Professional headshot
  - Brand colors and visual elements
  - Your logo
  - Brokerage logo
  - Website URL
  - Business email
  - QR code (optional but recommended)
- **QR code in Canva:** Left bar → Apps icon → search "QR code" → enter the URL (e.g., `gogosrealestate.com/partner`) → generates instantly → drag onto design.
- **Multi-platform resizing (Canva Pro):** Three dots icon → "Resize page" → search platform (YouTube, LinkedIn, Facebook, etc.) → Canva outputs the correct dimensions. Adjust layout manually after resize if needed.
- **Download:** Always export as **PNG**. Share → Download → select PNG → choose page → Download.
- **Template customization:**
  - Change colors: click element → color picker in top bar
  - Swap images: right-click → "Detach image" → Uploads → drag replacement image into frame
  - Edit text: double-click → type new content → change text color via top bar
