# Social Media Posting Schedule (Gogo Bethke)

> Gogo's strategic and philosophical walkthrough of the Mon–Sun posting schedule — including the no-scheduling-tools rule, price-equals-buyer-quality principle, Instagram workaround via ManyChat, Saturday/Sunday open house posting, and Friday timing rationale. Complements Kristy Waker's technical BoldTrail build video.

## Key Takeaways

- **Never use third-party scheduling tools.** Algorithm penalizes pre-scheduled posts — views are not even comparable to posting in the moment from your phone.
- **Never use Meta's own scheduling either.** Same problem. "The algorithm gods don't like it." Post live, always.
- **The schedule exists so you never run dry.** Inspiration failure → no post → no results → no commissions → broke. The schedule eliminates the gap.
- **Price = buyer quality:** Post a $100K listing → attract $100K buyers ($3K commission). Post a $1M listing → attract $1M buyers ($30K commission). Your posting choices directly control your income level.
- **Niche alignment:** Don't post a horse farm if you sell city condos. Post only what's in your niche — the listing you post attracts the buyer who wants it.
- **Instagram feed workaround for listings:** Instagram doesn't support links in the feed. Solution: set up a "buyer" ManyChat keyword. Post the listing image → caption says "comment BUYER and I'll send you the link." ManyChat delivers the link via DM automatically. Update the ManyChat link weekly to match the current listing.
- **Just-sold posts:** Make them fun. Instead of stating facts, create curiosity: "What if I told you this sold in 7 days for $179K? Comment VALUE if you want to know what YOUR home is worth."
- **Friday timing rationale:** Gogo moved the home-value post to Friday after her first attempt generated 27 home evaluation requests overnight — too many to handle on a weekday. Friday gives the weekend to review CMAs and follow up properly.
- **Saturday/Sunday:** Be at open houses. Post live, do a property tour video, or go live from the open house. The weekend is active selling time.
- **One-to-many principle:** One post can reach 1 person or 10,000. Once it's live, it works forever — someone can find it in the middle of the night, next week, in six months.
- **Morning habit:** Wake up, morning coffee, post done by 9am. That post works for you all day. You're done with marketing before most people start work.

## Full Schedule

| Day | Post | Notes |
|-----|------|-------|
| Monday | Any listing (your niche/price point) | Link on Facebook/LinkedIn/email/stories; use "buyer" keyword for Instagram feed |
| Tuesday | Your own listing OR just-sold OR any listing | Make just-sold posts fun; use "value" keyword CTA |
| Wednesday | Search for homes (KV Core IDX link) | Copy buyer search page URL; always collects visitor info |
| Thursday | Open houses this week (squeeze page link) | County-level if city is too small; evergreen squeeze page auto-updates |
| Friday | How much is your home worth (home value link) | Generates home eval requests; weekend gives time to review CMAs |
| Saturday | At open house — post live/tour | Active selling + content creation simultaneously |
| Sunday | At open house — post live/tour | Active selling + content creation simultaneously |
