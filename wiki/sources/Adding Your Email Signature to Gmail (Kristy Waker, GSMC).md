# Adding Your Email Signature to Gmail (Kristy Waker, GSMC)

> Kristy Waker walks through the full Gmail signature setup process — uploading the Canva image, adding clickable text links, setting signature defaults, and introduces WiseStamp as a free alternative for a fully clickable branded signature with social icons. Part 2 of a two-video sequence.

## Key Takeaways

### Gmail Setup — Step by Step
1. Download signature image from Canva as **PNG or JPEG** (not PDF)
2. Gmail → gear icon → **See all settings** → General tab → scroll to Signature
3. Click **Create new** → name your signature
4. Type clickable info above the image: name, title, email (as `mailto:` hyperlink), phone, website
5. Hit Enter → click **Insert image** icon → upload from Google Drive
6. **Important:** Image must be shared publicly in Google Drive ("anyone with link can view") before Gmail will accept it
7. Resize image if too large (click image → select "Large")
8. Add brokerage disclaimer text below if required
9. Go to **Signature defaults** → assign signature to "New emails" AND "Reply/forward" → **Save Changes**

**Common mistake:** Creating the signature but not assigning it to defaults → it never appears on emails.

### The Clickable Links Problem
A Canva PNG image is not clickable. Solution: type the key details (email address, website) as **hyperlinked text above the image** in Gmail's signature editor. The image handles the visual brand; the text handles the click functionality.

**`mailto:` trick:** Format email link as `mailto:yourname@email.com` — this makes clicking the email address open a new draft automatically.
