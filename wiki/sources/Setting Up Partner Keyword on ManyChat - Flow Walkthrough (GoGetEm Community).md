# Setting Up Partner Keyword on ManyChat — Flow Walkthrough (GoGet'Em Community)

> Sam (Gogo's tech team) walks through every step and branch of Gogo's "partner" keyword ManyChat flow — what triggers it, what each message says, and what the 23-hour follow-up does. Companion to the existing full build-from-scratch training (linked in lesson description).

## Key Takeaways

- **Trigger:** Comment or DM containing the word "partner" on any post → flow starts immediately
- **Step 1 message:** *"I am so excited you are interested in changing brokerages and maybe joining me at eXp."* → button: **"I am in"**
- **Step 2 (after "I am in" click):** *"Perfect. Jump into my calendar and let's talk."* → button: **"Partner with me"** (contains link to partner page — sign-up and next steps)
- **23-hour follow-up** (if "Partner with me" button was NOT clicked): *"Hey, did you get a chance to jump into my calendar yet? Let me know below."*
  - Reply **"Yes, I did"** → congratulatory message
  - Reply **"Not yet"** → prompted to book a call
- **Tone rule:** Every message must sound like how you normally speak — mannerism and tone must feel human, not like a bot
- **Cloneable template** available in lesson description — clone → swap out messages → done
- **Full build-from-scratch training** also linked in lesson description
- **ManyChat account required** — link to sign-up in lesson description

## Quotes

> "You should create a message in the normal way that you speak to people. So the mannerism and the tone has to sound like you, like the normal way you speak."
