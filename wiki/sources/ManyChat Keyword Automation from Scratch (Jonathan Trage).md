# ManyChat Keyword Automation from Scratch (Jonathan Trage)

> Full ManyChat Pro tutorial: building a keyword automation from scratch for Instagram comments and DMs, including public replies, buttons, and live testing.

## Key Takeaways

- Requires **ManyChat Pro** account with Instagram and Facebook accounts integrated
- **Three ways to start**: from scratch, via AI (answers questions → suggests automation), or from templates (e.g. grow SMS list, launch product, Instagram → WhatsApp)
- **Two triggers** (always set both for full coverage):
  1. "User comments on my post or reel" → All Posts or Reels (Pro feature) → keyword: e.g. "partner"
  2. "User sends a message" → detect specific word → "message IS [keyword]" (exact match only)
- **Public reply setup**: set 3 reply variations (ManyChat rotates them to avoid spam detection) — can paste same message in all 3 or vary them
- **Action**: Send Instagram message with text + emoji + button → "Open Website" → paste URL
- Use AI "Improve" button to: make more engaging, simplify language, rewrite friendly
- **Preview test**: ManyChat generates a 3-digit code → DM it to your own Instagram → triggers the automation flow for live testing
- Click "Set Live" to activate
- Frequency: set up once, Gogo's team prepares it BEFORE the post goes live so automation is ready when she posts
