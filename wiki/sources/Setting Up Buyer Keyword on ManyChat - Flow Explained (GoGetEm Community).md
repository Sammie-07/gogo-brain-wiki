# Setting Up Buyer Keyword on ManyChat — Flow Explained (GoGet'Em Community)

> Tutorial by Sam (Gogo's tech team) explaining the three elements of the ManyChat "buyer" keyword automation flow — not a build-from-scratch walkthrough, but a conceptual breakdown of what each element does.

## Key Takeaways

- The buyer keyword flow has exactly **3 elements**: Trigger → Condition → Message
- **Trigger**: Anyone who comments the word "buyer" on any post, OR sends a DM containing "buyer," starts the flow
- **Condition**: Checks whether the person follows the account
  - Follows → receives the message with the buyer link
  - Does not follow → receives "please follow to unlock the link" message; loop repeats until they follow
- **Message**: Contains the buyer website link; must sound like the agent — not robotic, not obviously automated
- A **cloneable template** is available in the GSMC lesson description — clone it, swap the message text, done
- A **full build-from-scratch training video** is also linked in the lesson description for those who want to understand every step
- A **Manychat account** is required to start; link in lesson description
- **Done-With-You program** provides tech support calls for agents who get stuck on setup

## Quotes

> "Make it sound like you, make it sound like your tone, mannerism, or whatnot so the person interacting with your DMs will not be able to find out that you use the bot."

> "However, if the person does not follow Gogo and they click this button, the condition will check again and still give them this particular message until they actually follow her."
