# GoHighLevel Automation — Comment Keywords (Jonathan Trage)

> GoGet'Em Community tutorial on setting up Instagram/Facebook comment keyword automations inside GoHighLevel — for agents who don't have ManyChat.

## Key Takeaways

- GHL keyword comment automation is the ManyChat alternative for agents already on GoHighLevel
- **Path**: Sub-account → Automation → Workflows → Create from Scratch → Trigger: "Instagram Comments on a Post"
- **Case sensitivity**: GHL IS case-sensitive — must enter keyword in 3 variations: lowercase, UPPERCASE, and Capitalized (e.g., partner, PARTNER, Partner)
- **Action**: Send Instagram DM with message + link (no button CTA option — GHL limitation vs ManyChat)
- Optional: "Track first level comments only" toggle — leave off to capture nested comments too
- **Tip**: name the workflow the same as the trigger for organization
- To add more keywords: clone the workflow → rename → update trigger keyword + DM message
- Must click "Publish" to go live
- **Testing**: have a team member comment the keyword on your Instagram, verify they receive the DM
- Works for both Instagram and Facebook with same setup

## Quotes

> "You see ManyChat offers more options, but if you don't have ManyChat and you want a workaround and you have GoHighLevel, that will be the best option."
