# How to Create a Calendar Event on GoHighLevel (Jonathan Trage)

> Full walkthrough of creating and customizing calendar events inside GoHighLevel — including availability, forms, email notifications, and reminders.

## Key Takeaways

- **Use Round Robin** even for solo users — fewer glitches than Personal Booking calendar type
- **Path**: Sub-account → Calendars → Calendar Settings → New Calendar → Round Robin
- **Setup fields**: name, team member, custom URL (should match name), meeting duration, availability (e.g. Mon-Fri 9-5)
- **Edit calendar** for advanced settings: logo, description, invite title (use custom variables like `{{contact.name}}`), Zoom link (connect Zoom account or paste URL directly), colors
- **Forms**: attach a custom form to the calendar — different forms for different calendar events
- **Confirmation page**: show thank you message OR redirect to a URL (e.g. Gogo redirects new partners to her partner page with 3 pre-call videos)
- **Auto-confirm**: must be ON — otherwise booking requires manual confirmation
- **Email notifications** (must enable individually): appointment booked, reminders (up to 3: e.g. 10 min, 1 hour, 1 day before), follow-up after call, cancellation/reschedule
- **Additional emails**: add assistant's email to receive copies of booking notifications
- **Minimum scheduling notice**: set to 1 day = no same-day bookings
- **Cancellation window**: set minimum hours before appointment for rescheduling/cancellation
- Replaces need for Calendly if already on GHL
