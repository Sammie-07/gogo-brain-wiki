# How to Create a QR Code in Canva (Ellen)

> Quick tutorial on generating QR codes directly inside Canva, including transparent background download for multi-platform use.

## Key Takeaways

- **Path**: Canva workspace → Left bar → Apps → search "QR code" → select the app
- Enter destination URL (where people land when they scan)
- Customize: background color, foreground color, margin size
- **Download with transparent background**: Share (top right) → Download → File type: PNG → check "Transparent background" → Download
- Transparent PNG can be placed on any design on any platform without a white box behind it
- Ellen is Gogo's lead designer
