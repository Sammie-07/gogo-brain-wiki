# Creating Professional Headshots Using AI

> Part of [[Gogo Bethke]]'s *Gogo's Social Media Challenge 2026* series. Presented by [[Laura Kurz]]. Step-by-step walkthrough for generating professional headshots using Higgsfield AI — no photographer required.

## Key Takeaways

- **Tool:** [Higgsfield](https://higgsfield.ai) — AI image generation with a personal avatar/twin system
- **Soul ID Character** is the feature used to create your avatar twin from your own photos
- Need **30–40 images** of yourself to train the avatar; selfies work, but Laura's preferred method is to record a video of yourself talking and screenshot frames — captures varied expressions, movements, and hand gestures naturally
- Download the **Higgsfield Reference Extension** (browser extension, found at the bottom of the Higgsfield site) to enable one-click recreation from any web image
- Use **Pinterest** as the style reference library — search "professional real estate headshot" (or any style/pose/background you want)
- Hover over any Pinterest image with the extension installed → green **Recreate** button appears → select your avatar → set aspect ratio **9:16** → Generate
- View all generated headshots in Higgsfield under **Browse Assets**
- Result: professional-quality headshot in any style, pose, or setting — ready for social profiles, brand tracker, and marketing materials
