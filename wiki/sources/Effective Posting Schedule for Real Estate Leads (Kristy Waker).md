# Effective Posting Schedule for Real Estate Leads (Kristy Waker)

> Full walkthrough of Gogo's 5-day weekly posting schedule and the exact BoldTrail/KVCore IDX link generation process for each day's post. Part of *Gogo's Social Media Challenge 2026*.

## Key Takeaways

**The 5-day schedule (built on IDX lead capture):**
- **Monday:** Any listing — find a property in your target price range and area, grab the direct link, post to Facebook
- **Tuesday:** Your own listing or just-sold (if MLS allows); if neither, any listing or a market report — anything that captures contact info
- **Wednesday:** "Search for Homes" — Multi-Property IDX squeeze page (evergreen link; auto-updates as new listings hit)
- **Thursday:** Open Houses — Multi-Property IDX squeeze page filtered to open houses (auto-updates; sparse Monday, fills by Thursday)
- **Friday:** "How Much Is Your Home Worth?" — Seller squeeze page (evergreen; never needs rebuilding)

**Wednesday/Thursday/Friday links are built once and reused forever.** Monday/Tuesday require finding a new listing each week.

**BoldTrail IDX Squeeze Page build process (Wed/Thu/Fri):**
1. BoldTrail → More → **Lead Engine** → IDX Squeeze Pages
2. Dropdown: select your website
3. Page type:
   - Wednesday/Thursday = **Multi-Property**
   - Friday = **Seller Squeeze**
4. Search criteria (left side of builder):
   - Location: choose county or city you serve (county recommended for rural/small markets)
   - Listing types: leave open or niche down (single-family, condo, new construction, etc.)
   - **Price minimum:** Always set one — e.g., $200K. What you post is the client you attract.
   - Price max: leave open
   - Options (just-listed, new construction, open house, etc.): **choose only ONE** — BoldTrail uses AND logic, so combining two options filters out most results
5. Page configuration (right side of builder):
   - Source: Facebook (tracks where leads came from in your CRM)
   - Hashtags: tag leads coming through this link (e.g., "homes for sale," "openhouse," "seller squeeze") — these are CRM database tags, not Instagram hashtags
   - Cover photo: use a high-priced listing — images are always sharp
   - **Property views before registration: set to 2** — lets nosy people look once, captures them as a lead on the second property view
6. Click **Generate Link** → copy → test in browser → paste into your posting schedule document

**Adding IDX pages to your website navigation:**
1. BoldTrail → Web & IDX → Manager → Custom Pages → Create Custom Page
2. Advanced Options → set name, category, and paste the squeeze page URL into the URL field
3. Add meta keywords → Save → Publish
4. Web & IDX → Manager → Website Settings → Custom Page Links → add the new page → Save
5. Page now appears in your website navigation — visitors can click directly into your IDX feed

**Posting tip:** After pasting a link into a Facebook post, wait for the image preview to populate, then **delete the URL text** — the image + link remain but the ugly URL disappears. Cleaner post, same lead capture.

## Quotes
