# Kirby Chan

Toronto-area ([[eXp Realty]]) listing agent and guest trainer in [[GoGet'Em Community]]; teaches ranking on Google for listing niches and getting **cited by AI** (ChatGPT / Gemini / Claude). Runs the "AI SEO Boot Camp," which doubles as a revenue-share recruiting funnel.

## Overview

Kirby Chan built a listing-first business by dominating a **smaller, winnable geography** (Richmond Hill, ~10k agents) instead of the saturated metro (Toronto, ~100k), and by engineering content so both Google and AI assistants recommend him. He credits the system for reaching eXp ICON faster and for scaling a co-sponsored downline.

## Key Facts

- **Listing-first, ~3 hrs/day, "book one listing appointment a day."** Reached eXp **ICON** in ~6 months (vs. 9 the prior year).
- Businesses: Kirby Chan & Co. Real Estate Team; **Team Forward** (his revenue-share org); **Kirby Chan Digital**.
- **AI SEO / GEO thesis:** win one niche + geography keyword cluster; Google's "Helpful Content" update lets a thin, genuinely-helpful site outrank Zillow/Redfin. Then use **"parasite SEO"** (daily "best agent in [city]" posts on LinkedIn / Rate My Agent) so ChatGPT/Gemini/Claude **cite you** when asked for the best agent.
- **AI content pipeline:** Claude writes scripts → HeyGen AI twin (or Captions.ai) voices them → AI editor adds B-roll (~5 min/video), run on separate **niche Instagram** accounts so the main brand stays real. Also teaches **foreign-language SEO** (write/voice content in a second language for near-zero competition).
- **Reviews:** drip ~1/week (never bulk — Google flags spikes), ask everyone including vendors, photo reviews rank stronger.
- **Attraction engine:** the "$15 AI SEO Boot Camp" recruits the #4/#5-ranked agent in *sister* cities → co-sponsor into eXp. Cited growth: 10 (2024) → 21 → 146 agents.
